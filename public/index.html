Cool API

Welcome to this very cool and very based tutorial where we will see how this very cool and very based API works

So, to start, this API was made by me and it's really simple to use:

Analytics API:

- /id/ | Generates a random 24-character ID with "RUBY_" at the start

- /counter/set?keyname=[anything] | If key doesn't exist, create it. If it exists, then make its number go up by one

- /counter/get?keyname=[anything] | Gets the value of a key without making its number go up by one

Other APIs:

- /news?country=CountryCode/ | Get the local news for any country (has a limit on requests)
